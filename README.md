# Agile CI Demo (FastAPI)

A tiny API to demonstrate **Agile**, **testing**, and **CI/CD** using GitHub.

## What students learn
- Agile user stories & acceptance criteria (Gherkin)
- Definition of Done
- Test-driven development (pytest)
- CI (lint + test + type check on PR)
- CD (tag -> container image on GHCR)

## Quick start
```bash
make install
make test
make run
# open http://localhost:8000/health
